# zapret-ui

Удобный графический интерфейс (Windows) для [`Flowseal/zapret-discord-youtube`](https://github.com/Flowseal/zapret-discord-youtube) — обхода DPI-блокировок Discord, YouTube и др.

Один исполняемый файл на Rust + [Slint](https://slint.dev/). Сам скачивает zapret, запускает `winws.exe` с выбранной стратегией или ставит её Windows-службой — без ручной возни с `.bat`.

## Возможности

- **Авто-загрузка zapret** с GitHub, если его ещё нет на ПК.
- **Все стратегии из дистрибутива** — пресеты читаются прямо из `.bat`-файлов (`general`, `general (ALT)` … `(ALT11)`, `FAKE TLS AUTO`, `SIMPLE FAKE` и т.д.) и парсятся в готовую команду `winws.exe`.
- **Запуск как процесс** (кнопка START) или **как Windows-служба** (автозапуск при загрузке).
- **Подхват** уже установленной копии или запущенного процесса/службы.
- **Проверка обновлений** zapret из репозитория upstream.
- Живые логи `winws.exe` с возможностью выделить и скопировать текст.
- Тёмная/светлая тема, сворачивание в трей.

> **Почему не GitHub API?** У многих провайдеров `api.github.com` заблокирован DPI (тем самым, что обходит zapret). Поэтому версия берётся с `raw.githubusercontent.com`, а архив — с `codeload.github.com` — оба доступны там, где API недоступен.

## Требования

- Windows 10/11 x64.
- Права администратора нужны для запуска обхода (winws использует драйвер WinDivert), теста стратегий и операций со службой Windows.
  - Для операций со **службой** UAC запрашивается по месту — отдельный elevated-процесс выполняет действие и завершается.
  - Для **запуска обхода как процесса** и **теста стратегий** запустите приложение от имени администратора (баннер вверху предложит перезапуститься с правами). Само окно при этом работает и без прав — недоступны лишь действия, которым нужен драйвер.

## Установка

Скачайте `zapret-ui.exe` из [Releases](../../releases) и запустите. Дополнительные DLL не нужны — всё в одном бинарнике.

## Использование

1. На вкладке **Home** нажмите **Install zapret** (если не установлен).
2. На вкладке **Strategies** выберите пресет (**Select**) — откроется главный экран.
3. Нажмите **START** (запуск как процесс) либо **Run as service** (как служба).
4. Если конкретный пресет не работает у вашего провайдера — попробуйте следующий `ALT`.

## Сборка из исходников

```powershell
# нужен стабильный Rust (https://rustup.rs)
cargo build --release
# результат: target\release\zapret-ui.exe
```

Запуск тестов:

```powershell
cargo test
```

## Стек

| Слой        | Технология                                  |
|-------------|---------------------------------------------|
| Язык        | Rust (stable)                               |
| UI          | Slint (winit backend, Skia renderer)        |
| Async/сеть  | tokio + reqwest (rustls)                     |
| Служба      | windows-service                             |
| Архивы      | zip                                         |

## Дисклеймер

Некоторые антивирусы помечают `winws.exe` из-за работы с сырыми сетевыми пакетами (WinDivert). Это ложное срабатывание; при необходимости добавьте папку установки в исключения. zapret-ui лишь оборачивает официальный дистрибутив `Flowseal/zapret-discord-youtube`.

## Лицензия

Код UI распространяется как есть. Запускаемый zapret и его компоненты принадлежат их авторам — см. [upstream-репозиторий](https://github.com/Flowseal/zapret-discord-youtube).
